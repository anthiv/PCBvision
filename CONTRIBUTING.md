# Contributing

Thank you for your interest in PCBVision!

At this stage, PCBVision is primarily a personal learning and portfolio project focused on Machine Learning Engineering and Industrial Computer Vision.

Suggestions, bug reports, and discussions are always welcome. If you would like to contribute, please:

1. Open an issue describing the proposed change.
2. Discuss the solution before starting implementation.
3. Follow the existing project structure and coding style.
4. Ensure any new functionality includes appropriate documentation and tests where applicable.

The primary goals of this repository are readability, maintainability, and learning through engineering best practices.
